# Hyper-Space

Welcome to HyperSpace, a futuristic web experience built to explore the intersection of immersive 3D graphics, generative AI, and modern web design. This project is more than just a website—it's a digital frontier designed to feel both beautiful and intelligent.

I wanted to see what was possible when you push the boundaries of frontend development. The result is a sleek, responsive, and interactive application that showcases the power of combining cutting-edge tools.

# Core Features
Interactive 3D Hero: The homepage features a stunning 3D object created with Three.js that subtly reacts to your mouse movements, creating an engaging and immersive welcome.
AI-Powered Content Suggester: Curious what to watch or read next? Tell our AI what you're interested in, and it will generate personalized recommendations for you, powered by Google's Genkit.
Modern, Responsive UI: Built with Next.js and styled with Tailwind CSS and ShadCN UI, the interface is designed to be sleek, fast, and fully responsive across all devices.
Smooth Animations: The entire site is filled with subtle, performance-optimized animations and parallax effects that add depth and a sense of futuristic polish.

# Tech Features
Framework: Next.js 15 (App Router)
3D Graphics: Three.js / react-three-fiber
Generative AI: Genkit
Styling: Tailwind CSS
UI Components: ShadCN UI
Deployment: Firebase App Hosting

# Getting Started
Ready to explore the code yourself? Here’s how to get it running locally:

Clone the repository:

```
git clone https://github.com/muhabdev0/Hyper-Space.git'''
cd hyperspace
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```

Open up http://localhost:9002 in your browser to see the magic happen.
